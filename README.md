# bouncingball
  Bouncing ball was an introductory lesson on manipulating the page with javascript.
  In this case, a <div> called "ball"  is having its style properties manipulated with Javascript.
  I used a basketball image and court background image to make it more fun.
  
# Installation
  Download the files all into the same folder and open index.html with your web browser.
